# Connect 4 Android Game – Individual UI Enhancements

## Description
This repository contains my individual enhancement work for a Connect 4 Android game originally developed as a group university project. My work focused on advanced UI/UX improvements, personalization features, and gameplay presentation enhancements.

## Focus of My Work
The purpose of this repository is to showcase the individual features and interface improvements I designed and implemented, rather than to present a fully standalone runnable application.

## Individual Enhancements
- Card-based UI improvements
- Improved button colour styling
- Dynamic player-colour button updates
- Avatar selection with visual highlighting
- Dark mode toggle
- Disc-drop animation
- Flashing winning-cell effect
- Winning banner popup
- Share-result functionality

## Technologies Used
- Java
- Android Studio
- Android SDK
- XML layouts
- SharedPreferences

## Project Context
These enhancements were developed as part of the individual component/report for the Mobile Application Development assignment. The accompanying report explains the design choices, research basis, implementation details, and results.

## Note
This repository highlights my individual feature development and UI enhancement work. Some files depend on the original group project structure for full execution.
